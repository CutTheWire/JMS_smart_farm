 기능 | 내용

`Ar` | 아두이노를 통해 펌프, 시스템 팬, LED를 릴레이 모듈로 제어</br>토양습도와 대기 온습도 데이터를 PC(RaspberryOS 또는 WindowsOS)로 전달
`DB` | 아두이노를 통해 받은 데이터를 파이썬을 통해 .DB 확장자 파일로 저장된 데이터 모음
`API` | 아두이노로 부터 받은 데이터를 전처리 및 릴레이 모듈을 제어할 수 있는 코드 작성됨
